# LMFNet
A Lightweight RGB-D Localization and Refinement Network for Salient object detection

# README
Please read the readme file, which contains everything we have described.

# Installation
First, we need to install a proper Anaconda environment, which is essential.

# Requirements
* python 3.11
* pytorch 2.0.1
* Numpy 1.25.1
* torchvision
* 
## RGB-D Training and Testing
We provide the content containing all the required datasets (including training and test sets). In the root directory of this repository, create a folder named 'datasets', and then copy and paste the content from the link into it. Now you will be able to run all the configurations.Link to the network datasets: https://pan.baidu.com/s/1b_J5Clsbr2p6ISzXdczChg 提取码: w8q6

# Run Test code
